# Automated Timetable Generator

An AI-based automated timetable generation system developed using Spring Boot, Java, Python, and MySQL. The project helps educational institutions generate optimized class timetables automatically while reducing manual scheduling conflicts.

## Features
- Automated timetable generation
- Subject and faculty allocation
- Conflict-free scheduling
- Database integration using MySQL
- Java Spring Boot backend
- Python-based timetable logic
- Easy timetable management

- HTML5
- CSS3
- JavaScript
- Bootstrap

- Spring Boot
- Python
